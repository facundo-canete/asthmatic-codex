# Expresiones

## A posteriori

### Definición:
Significa «Desde lo posterior» o «A partir de lo que viene después» y se refiere a un conocimiento, conclusión o razonamiento que se obtiene después de observar la experiencia, los hechos o los resultados; e.g., «*A posteriori*, nos dimos cuenta de que habíamos tomado una mala decisión» cf. A priori.


## A priori

### Definición:
Significa «Desde lo anterior». Se usa para referirse a una idea, conclusión o conocimiento que se obtiene antes de observar los hechos o la experiencia; e.g., «*A priori*, parecía una buena idea invertir en este proyecto»; cf. A posteriori.


## Ad hoc

### Definición:
Significa «Para esto» o «Para este propósito». Se usa para referirse a algo creado o adaptado específicamente para una situación particular. En derecho, política, informática y ciencia se usa mucho cuando alguien dice que una solución es «ad hoc», a veces de forma neutral y a veces con la crítica implícita de que es un parche hecho a medida y no una solución general; e.g., «La empresa armó un comité *ad hoc* para investigar el incidente».


## Alma mater

### Definición:
Significa literalmente «Madre nutricia» o «Madre que alimenta». Se usa para referirse a la institución educativa donde una persona estudió, especialmente una universidad. También puede usarse para describir a la persona que impulsa o inspira un proyecto, aunque ese uso es menos frecuente que el académico; e.g., «La Universidad Nacional de Córdoba es mi *alma mater*».


## Alter ego

### Definición:
Significa «Otro yo». Se usa para referirse a una segunda personalidad, identidad alternativa o persona con la que alguien tiene una identificación muy profunda. En psicología, literatura, cine y cómics se usa mucho para describir identidades alternativas; e.g., «En el escenario, ese personaje es el *alter ego* del cantante», Superman y Clark Kent pueden considerarse mutuamente *alter ego*.


## Carpe diem

### Definición:
Significa «Aprovecha el día» o, más literalmente, «Cosecha el día». Se utiliza para expresar la idea de disfrutar el presente y aprovechar las oportunidades sin preocuparse excesivamente por el futuro. La frase proviene de la obra *Odas* de Horacio y es una de las expresiones latinas más famosas; e.g., «Decidió viajar sin pensarlo demasiado: *carpe diem*».


## Conditio sine qua non

### Definición:
Significa «Condición sin la cual no». Se utiliza para referirse a un requisito indispensable o absolutamente necesario para que algo ocurra. Es muy común en derecho, filosofía, economía y lenguaje académico. En español suele encontrarse la forma abreviada no latina «Condición sine qua non», aunque el significado es exactamente el mismo; e.g., «La aprobación del presupuesto es una *conditio sine qua non* para iniciar el proyecto».


## Cum laude

### Definición:
Significa «Con alabanza» o «Con honores». Se utiliza para indicar que una persona obtuvo un título académico con un nivel destacado de rendimiento. En muchas universidades forma parte de una escala de distinciones académicas que también incluye *magna cum laude* que significa «Con grandes honores» y *summa cum laude* que significa «Con los más altos honores»; e.g., «Se graduó *cum laude* en ingeniería».


## De facto

### Definición:
Significa «De hecho». Se utiliza para describir una situación que existe o funciona en la práctica, aunque no esté reconocida oficialmente o por ley. Suele aparecer en contraste con *de iure* que significa «De derecho» o «Legalmente»; e.g., «Aunque no tiene el cargo formal, es el líder *de facto* del grupo».


## Grosso modo

### Definición:
Significa «De manera aproximada» o «A grandes rasgos». Se utiliza cuando se quiere explicar algo sin entrar en demasiados detalles. Aunque es una expresión latina, hoy se usa mucho en español culto y académico para indicar que una explicación es una simplificación general y no un análisis exhaustivo; e.g., «*Grosso modo*, el proyecto consiste en modernizar el sistema y reducir costos».


## In fraganti

### Definición:
Significa «En el momento en que ocurre el hecho». Se utiliza para indicar que alguien fue sorprendido mientras cometía una acción, generalmente un delito o una falta. La forma latina completa es *in flagranti delicto* que significa «En el delito ardiente»; e.g., «La policía lo atrapó *in fraganti* mientras intentaba forzar la cerradura».


## In situ

### Definición:
Significa «En el lugar» o «En el sitio». Se utiliza pra indicar que algo ocurre, se observa o se estudia en su ubicación original. Es muy común en arqueología, biología, medicina, geología e investigación científica en general, donde suele contrastarse con estudios realizados en laboratorio o sobre muestras trasladadas; e.g., «Los arqueólogos analizaron las piezas *in situ* antes de retirarlas».


## Ipso facto

### Definición:
Significa «Por el hecho mismo» o «Automáticamente consecuencia de ello». Se utiliza para indicar que una cosa deriva de otra de manera inmediata y directa. Es muy utilizada en textos jurídicos y formales para señalar que una consecuencia se produce de forma automática, sin necesidad de una acción o decisión adicional; e.g., «Al renunciar al cargo, quedó *ipso facto* excluido de la comisión».


## Modus operandi (M.O.)

### Definición:
Significa «Modo de operar» o «Forma de actuar». Se utiliza para describir el método habitual que una persona, grupo u organización emplea para realizar una actividad; e.g., «La policía identificó al sospechoso por su *modus operandi*, que coincidía con otros robos».


## Quod erat demonstrandum (Q.E.D.)

### Definición:
Significa «Lo que se quería demostrar». Se utiliza al final de una demostración matemática o lógica para indicar que se ha probado correctamente la proposición planteada. Durante siglos fue habitual terminal las demostraciones con Q.E.D.. Hoy muchos matemáticos prefieren usar el símbolo ∎ (llamado «tumba», «tombstone» o «Halmos box»), aunque la abreviatura sigue siendo ampliamente reconocida; e.g., «Tras completa la demostración escribió: "*quod erat demonstrandum*"».


## Statu quo

### Definición:
Significa «Estado en que». Se utiliza para referirse a la situación actual de algo, especialmente cuando se habla de mantenerla sin cambios. La expresión completa era *In statu quo ante* que significa «En el estado en que estaba antes», pero con el tiempo quedó simplemente *statu quo*, quye hoy se utiliza en política, derecho, economía y en el habla cotidiana para referirse al estado de cosas vigente; e.g., «El acuerdo busca preservar el *statu quo* hasta que se llegue a una solución definitiva».


## Vox populi

### Definición:
Significa «Voz del pueblo». Se utiliza para referirse a una opinión, creencia o información ampliamente conocida o compartida por la gente. Suele emplearse para señalar algo que todo el mundo parece saber, aunque no necesariamente exista una confirmación oficial. De ahí la famosa expresión *Vox populi, vox Dei* que significa «La voz del pueblo es la voz de Dios».
